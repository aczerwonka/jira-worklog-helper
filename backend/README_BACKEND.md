"Jestem w trakcie tworzenia backendu w Spring Boot 3. Aplikacja ma integrować się z Jira API oraz Google Calendar API. Baza PostgreSQL będzie przechowywać reguły mapowania zdarzeń z kalendarza na taski w Jirze. Używamy OAuth2 do autoryzacji."
Aplikacja ma umożliwać wpisywanie hidden worklog time w ticketach jira.
